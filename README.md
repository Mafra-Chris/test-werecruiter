# test-werecruiter

### Instalação
Abra um terminal e execute os seguintes comandos
```
cd frontend
```

```
npm install
```

### Execução
#### Vite
No mesmo terminal execute:
```
npm run dev
```
#### JSON Server
Para rodar a api fake abra outro terminal e execute os seguintes comandos:

```
cd frontend
```

```
json-server --watch db.json 
```
